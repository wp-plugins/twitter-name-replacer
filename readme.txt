Contributors: spitblog
Tags: twitter, username, post, replace
Requires at least: 2.0
Tested up to: 2.7.1
Stable tag: trunk

Replaces Twitter usernames (@spitblog) with correctly linked ones (a href="https://twitter.com/spitblog")

== Description ==

Whenever you wanted to refer to another Twitter user, you had to write his/her username and manually link to the corresponding twitterpage.

Now let the Twitter Name Replacer do that for you.

* @spitblog will be displayed as <a href="https://twitter.com/spitblog">@spitblog</a>
* !@spitblog will be displayed as @spitblog (without link)
* mail addresses will be recognized and skipped

== Installation ==

1. Upload `twitter-name-replacer.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
